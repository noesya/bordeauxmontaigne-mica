Téléchargez le logo du MICA

  • Pour télécharger un logo en PDF, cliquez sur son aperçu.
  • Pour télécharger un logo en PNG, effectuez un clic droit sur son aperçu puis, dans le menu qui apparaît, cliquez sur "Enregistrez la cible du lien sous"
  • Pour une demande spécifique, contactez le Webmaster du MICA : web.mica@msha.fr

Version PDF

Cette image vectorielle très haute résolution en CMJN est à utiliser dans les documents destinés à l’impression.

L’espace entre les carrés étant transparent, le logo peut donc être placé sur n’importe quel fond de couleur.

Composition des couleurs :
Violet : Cyan 90 / Magenta 90
Orange : Magenta 55 / Jaune 90
Rouge : Cyan 15 / Magenta 100 / Jaune 60
Vert : Cyan 50 / Jaune 100

Version PNG

Cette image bitmap en RVB (1500 pixels de largeur) est à utiliser sur les sites Internet, les fichiers Word, PowerPoint…

L’espace entre les carrés étant transparent, le logo peut donc être placé sur n’importe quel fond de couleur.

Composition des couleurs :
Violet : #37388b
Orange : #f08b25
Rouge : #cd0247
Vert : #97bf0d

Versions avec texte

Mêmes caractéristiques que pour les versions sans texte.
